# cs_baby_wishlist

Modulo personalizado de Odoo para gestionar listas de nacimiento desde la web.

## Funcionalidades principales

- Creacion y gestion de listas por usuarios portal.
- Enlace publico para compartir la lista.
- Enlace privado para gestion de la lista.
- Flujo de compra desde la lista publica con control de cantidades.
- Actualizacion automatica de cantidades compradas al confirmar pedidos.
- Notificaciones por correo para creacion y actualizaciones de compra.

## Requisitos

- Odoo 19
- Modulos: `website`, `website_sale`, `portal`

## Instalacion

1. Copiar el modulo en la ruta de addons personalizados.
2. Actualizar lista de aplicaciones en Odoo.
3. Instalar `cs_baby_wishlist`.
